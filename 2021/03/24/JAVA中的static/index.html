静态方法

静态方法是一种不能向对象实施操作的方法。它不能使用任何类的对象，换句话说没有隐式的参数。但是静态方法可以访问自类中的静态域。如：

我们可以通过类名调用这个方法：

int n = Employee.getNextId();

静态方法的使用主要出现在一下两种情况：

  1. 一个方法不需要访问对象状态，所需的参数都是显示参数提供的。
  2. 一个方法只需访问类的静态域。
