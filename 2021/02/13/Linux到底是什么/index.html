Linux起源

Linux全称GNU/Linux （这很重要），它是一种类UNIX的操作系统，其内核是由林纳斯·本纳第克特·托瓦兹（Linus Benedict Torvalds，1969年12月28日- ）写的。

20世纪80年代，当时可供计算机选用的操作系统主要有Unix。但是它并不是面向我们这些普通人的，Unix主要的用户仅仅是那些大厂，大公司。但这时林纳斯站出来了，他在大学里仅仅花了一年的时间就开发出了Linux的内核，并使之开源。Linux继承了Unix以网络为核心的设计思想，是一个性能稳定的多用户网络操作系统 。

Linux的各种发行版

Linux有上百种不同的发行版 。列如（Ubuntu, CenntOS, RedHat, Kail…）。就好像安卓有华为，小米， oppo, vivo。

主流发行版

  • Debian(其派生版本有Ubuntu, Linux Mint, Kail)
  • Fedore(其派生版本有RedHat, CentOS, Xange, Yellow Dog Linux)
  • openSUSE(其派生版本有SUSE Linux Enterprise )

Linux的四大组成部分

  1. Linux Kernel 内核
  2. GNU工具 GNU（GNU’s Not Unix! ）
  3. GUI Desktop环境
  4. Application 应用

GUI和GNU-Linux的系统组成

有关Linux的系统组成，请参阅下面的思维导图。

Linux内核

  1. 硬件设备 管理使用
  2. 软件程序 (系统) -> 操作软件
  3. 系统内存
  4. 文件管理（保存文件， 删除文件， 修改文件）

文件系统

简单的来讲文件系统就是文件读和写的标准。

linux支持的文件系统类型：ext, ext2, ext3, ext4, hpfs, isp9660, JFS…

Windows支持的文件系统类型： FAT32, NTFS, exFAT

不同类型的文件系统产生不同的效果 。

GNU

GNU组织

Unix上具有的一些软件，Linux内核本身没有，所以GNU就模仿Unix，为Linux写了一些必要的软件。

GNU核心

coreutils

它是供Linux使用的工具。它有三个组成。

  1. 用来处理文件的工具
  2. 用来操作文本的工具
  3. 用来管理进程的工具

shell

它是提供给用户使用的软件： 用户拿它来使用电脑，并且与电脑交互。

shell(命令行壳层)提供一个命令行界面(CLI); 而图形壳层提供一个图形用户界面(GUI)。

bash shell 基础shell

其他的shell: zsh, ash, korn, tcsh
