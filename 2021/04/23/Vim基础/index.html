简单介绍

Vim是从vi发展出来的一个文本编辑器。其代码补完、编译及错误跳转等方便编程的功能特别丰富，在程序员中被广泛使用。和Emacs并列成为类Unix系统用户最喜欢的编辑器。

四种模式

  1. 普通模式（Esc）
  2. 插入模式（i, a, o）
  3. 命令模式（:）
  4. 视图模式（v, V）

使用

移动

取代普通的方向键或是鼠标，h 是左， j是下， k是上，l 是右。

翻页

翻整页： Ctrl + f即Pagedown, Ctrl + b即PageUp

翻半页： Ctrl + d, Ctrl + u

翻到最后一行：shift + g即直接输入大写G。

翻到第一行：gg

翻到指定的行：ng n为行号。

不同方式编辑文本，跳跃单词

编辑：

i 光标位置前面插入

a光标位置后面插入

o直接下一行插入

x删除光标所在字符

dd删除整个一行

u撤销

dw删除所在光标所在的单词

b跳越单词首字母（往前跳跃）

e跳跃单词最后

w跳跃下一个单词的首字母

shift + w,shift + b,shift + e大跳

r替换一个

R替换任意

跳跃行首行尾

shift + 6跳到行首

shift + 4跳到行尾

大括号跳跃函数段落

{}

p粘贴

y复制

y + w复制这个单词

y + $从当前复制到末尾
